#Pass.py is a rudimentary password cracker that is able to brute force 
#sha256 hashes given a set of predetermined rules

:Running instructions:
In order to run, please place hashlib.py and words.txt(standard linux dictionary) in the root directory of the pass.py script. Once running, simply follow the onscreen prompts. Successful outputs will be written to 'Solved.txt' in the scripts same directory and will also be outputted to console.\
